##  Overview

This project involves the mechanical design and prototyping of a **pediatric cystoscopic grasper** intended for minimally invasive urological procedures. The development focuses on achieving reliable functionality within strict pediatric anatomical constraints while maintaining surgeon ergonomics and manufacturability.

Work is conducted in close collaboration with clinical stakeholders and medical device manufacturers.

##  Clinical Collaboration

This project is being developed in close collaboration with:

* Senior Urologist (clinical lead)
* Supervising Urologist

The collaboration ensures that the device design is continuously informed by real surgical workflow requirements, pediatric anatomical constraints, and clinical usability considerations.

##  Development Goals

* Develop a reliable minimally invasive grasping mechanism for pediatric use
* Ensure ergonomic handling for surgical users
* Address miniaturization constraints for pediatric anatomy
* Enable manufacturable and scalable design
* Align development with ISO 10993 and ISO 13485 expectations
* Support ongoing patent activities
